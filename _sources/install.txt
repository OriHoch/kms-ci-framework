Installation guide
==================

Prerequisites
-------------

Just php, version 5.3.28 or later.

Installation
------------

Clone the kms-ci-framework repository::

    $ git clone https://github.com/kaltura/kms-ci-framework.git

Put the bin/kmsci executable in your PATH.

Optional
--------

Depending on which features of kms-ci-framework you are going to use you might need the following tools on your PATH:

* phantomjs - `PhantomJS <http://phantomjs.org/>`_.
* casperjs - `CasperJS <http://casperjs.org/>`_.
* phpunit - `PHPUnit <http://phpunit.de/>`_.
